DevHub_grunt
============

DevHub用の grunt 設定ファイル

## grunt のインストール

```
npm install -g grunt-cli
```

## プロジェクトの作成

```
mkdir DevHub_grunt
cd DevHub_grunt
npm init
npm install grunt --save-dev
npm install grunt-contrib -save-dev
```

## Gruntfile.js の作成

```
vi Gruntfile.js
```

## gruntの実行

```
grunt
```
